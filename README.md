# My Work（Beginner Portfolio Site）

CodeJump初級編の課題として、フォトグラファーのポートフォリオサイトを再現実装しました。  
Flexboxによるレイアウトと、レスポンシブ対応の理解を目的としています。

---

## 使用技術
- HTML5
- CSS3（Flexbox）
- レスポンシブ対応（@media screen and (max-width: 600px)）

---

## 実装内容
- Flexboxによる作品一覧（ul / li）のレイアウト構築
- PC表示での横並びレイアウト
- SP表示での縦並びレイアウト（flex-direction: column）
- width: 100% を用いたモバイル最適化
- コンテンツ幅と余白の調整

---

## 学んだこと
- Flexboxは親要素に指定することでレイアウトが制御できることを理解した
- PCとスマホでレイアウトを切り替える基本的な考え方を学んだ
- flex-direction の使い分けによって表示が変化することを理解した
- HTML構造とCSS指定（クラス・ID）の関係を整理できた
